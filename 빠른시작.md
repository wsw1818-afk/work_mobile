# 🚀 가계부 네이티브 - 빠른 시작 가이드

## ⚡ 가장 빠른 방법

### 1️⃣ build-apk.bat 더블클릭

프로젝트 폴더에서 **build-apk.bat** 파일을 더블클릭하세요.

### 2️⃣ 로그인

Expo 계정이 없다면:
- https://expo.dev/signup 에서 2분 안에 생성
- 이메일 또는 GitHub 계정 사용 가능

터미널에 이메일/비밀번호 입력하세요.

### 3️⃣ 대기 (15-20분)

클라우드에서 자동으로 빌드됩니다.
- ☕ 커피 한 잔 하세요
- 💻 다른 작업을 해도 됩니다
- 📊 진행 상황 URL이 표시됩니다

### 4️⃣ 다운로드

빌드 완료 후:
- 터미널에 표시된 다운로드 링크 클릭
- 또는 https://expo.dev → Projects → gagyebu-native → Builds

### 5️⃣ 설치

**방법 A: 휴대폰에서 직접 다운로드 (권장)**
1. 휴대폰 브라우저에서 다운로드 링크 열기
2. APK 자동 다운로드
3. 파일 탭하여 설치

**방법 B: PC → 휴대폰 전송**
1. PC에서 APK 다운로드
2. USB 케이블로 휴대폰 연결
3. Download 폴더로 복사
4. 휴대폰에서 파일 관리자로 설치

---

## ⚙️ 수동 명령어

터미널에서 직접 실행하려면:

```bash
# 프로젝트 폴더로 이동
cd "D:\claude_CLI\work\모바일2\gagyebu-native"

# 로그인 (최초 1회)
eas login

# 빌드 시작
eas build --platform android --profile preview
```

## ⚠️ 문제 해결

### "알 수 없는 출처" 오류
설정 → 보안 → 알 수 없는 출처 허용

### "EAS CLI가 없습니다"
```bash
npm install -g eas-cli
```

### 로그인 실패
1. https://expo.dev에서 계정 확인
2. 비밀번호 재설정
3. `eas logout` 후 재로그인
