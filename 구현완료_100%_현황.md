#### 시나리오 1: 기본 거래 흐름
1. 앱 실행 → Dashboard 확인 (빈 상태)
2. '+' 버튼 → 거래 추가 화면
3. 금액: 10,000원, 카테고리: 식비, 결제수단: 신용카드
4. '지출 추가' 버튼 클릭
5. Dashboard로 돌아가서 통계 확인
   - 총 지출: 10,000원
   - 카테고리별 지출: 식비 100%
   - 최근 거래: 방금 추가한 거래 표시

#### 시나리오 2: 예산 관리
1. Budgets 탭 이동
2. '+' 버튼 → 예산 추가
3. 카테고리: 식비, 예산: 300,000원
4. 예산 카드 확인
   - 사용: 10,000원 / 300,000원
   - 진행률: 3.3%
   - 프로그레스바 표시

#### 시나리오 3: 반복 거래
1. Recurring 탭 이동
2. '+' 버튼 → 반복 거래 추가
3. 이름: 넷플릭스, 금액: 9,500원, 주기: 매월, 일: 1일
4. 거래 카드 확인
   - 활성화 토글 동작 확인

#### 시나리오 4: 자동 분류 규칙
1. Rules 탭 이동
2. '+' 버튼 → 규칙 추가
3. 패턴: "스타벅스", 카테고리: 카페/간식
4. 규칙 활성화
5. 거래 추가 시 "스타벅스" 입력하면 자동으로 카테고리 적용
